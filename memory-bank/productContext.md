# KYCGate — Product Context

## Why This Project Exists
Web3 games need a way to verify player identities before granting access. This platform provides a complete KYC onboarding flow with admin review, ensuring only verified users can access the gaming platform.

## Problems Solved
1. **Identity verification**: Multi-step KYC with document upload and wallet linking
2. **Admin control**: Centralized review panel for approving/rejecting applications
3. **Access gating**: Game URL only accessible to approved users
4. **Compliance**: Government ID collection and storage with secure S3 handling

## User Experience Goals
- Simple, guided KYC flow (4 clear steps)
- Visual feedback at every stage (status tracker, upload progress, wallet connection state)
- Dark, gaming-aesthetic UI that feels modern and trustworthy
- Mobile-friendly for document upload from phone cameras
- Clear communication of application status and next steps
